# Customer Behavior Analysis Using SQL

## Project Overview
This project analyzes customer purchasing behavior using SQL to generate actionable business insights. The analysis focuses on revenue trends, customer segmentation, product performance, discount usage, and subscription impact.

The goal is to help businesses understand customer patterns and improve decision-making using data-driven insights.

---

## Tools & Technologies
- MySQL Workbench
- SQL (MySQL)
- Data Analysis Techniques

---

## Key Objectives
- Analyze customer purchasing behavior
- Identify high-value customers
- Study impact of discounts and subscriptions
- Segment customers based on buying behavior
- Find top-performing products and categories

---

## Key Analysis Performed

### 1. Revenue Analysis
- Total revenue by gender
- Revenue by age group
- Subscription vs non-subscription revenue

### 2. Customer Behavior
- Customer segmentation (New, Returning, Loyal)
- Repeat buyer analysis
- High-value customer identification

### 3. Product Performance
- Top 5 highest rated products
- Top 3 products per category
- Discount usage per product

### 4. Business Insights
- Shipping type vs spending behavior
- Discount impact on sales
- Customer purchase trends

---

## SQL Concepts Used
- Aggregate Functions (SUM, AVG, COUNT)
- GROUP BY & HAVING
- CASE Statements
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions (ROW_NUMBER)

---

## Key Insights
- Identified loyal and high-value customer segments
- Found top-performing products and categories
- Analyzed how discounts affect purchasing behavior
- Discovered revenue patterns across different customer groups
